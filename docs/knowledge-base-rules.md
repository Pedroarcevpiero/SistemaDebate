# Reglas de la base de conocimiento

Cada debate produce una **fuente de conocimiento reutilizable** en
`knowledge/debate-knowledge-[fecha]-[slug].md`. Estas reglas rigen cómo se construye y
se reutiliza.

## Propósito

La base de conocimiento permite que debates, investigaciones y decisiones futuras
reutilicen lo aprendido sin repetir todo el debate. El archivo de conocimiento es más
breve que el debate completo y está escrito para ser consultado después.

## Estructura del archivo de conocimiento

```
# Fuente de conocimiento — [tema]
## Tesis debatida
## Contexto
## Conclusión principal
## Ganador del debate
## Argumentos fuertes a favor
## Argumentos fuertes en contra
## Evidencia útil para reutilizar
## Riesgos o advertencias
## Lecciones aprendidas
## Preguntas abiertas
## Cuándo reutilizar esta fuente
```

## Reglas de escritura

- Más breve que el debate completo; sin transcripción de rondas.
- Solo conclusiones, evidencia útil y lecciones.
- No inventar fuentes; reutilizar solo evidencia realmente usada en el debate.
- Marcar preguntas abiertas y puntos no resueltos.
- Indicar cuándo conviene reutilizar la fuente (qué tipo de tesis futura).

## Reglas de reutilización

- Antes de un nuevo debate, revisar `knowledge/` por temas relacionados.
- Tratar el conocimiento previo como insumo, no como veredicto definitivo.
- Si nueva evidencia contradice una fuente previa, prevalece la evidencia nueva y se
  documenta el cambio.
